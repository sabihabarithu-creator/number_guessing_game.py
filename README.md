# 🎯 Number Guessing Game

A simple Number Guessing Game built using Python. The computer randomly selects a number between 1 and 100, and the player has to guess it using the hints provided.

## 🚀 Features

- Random number generation
- User-friendly interface
- Hints for each guess (Too High / Too Low)
- Counts the number of attempts
- Simple menu-free gameplay

## 🛠 Technologies Used

- Python 3
- Built-in `random` module

## 📂 Project Structure

```
number-guessing-game/
│── number_guessing_game.py
│── README.md
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run:

```bash
python number_guessing_game.py
```
